# Q
对需要改变或者想要思考的东西通过元认知对思考进行分析认知迭代并记录
1。 对拖延的思考
我在思考我为什么会有拖延这一行为说实话它真的害我不浅，好像自我存它就如影随形，小学被拖延的任务作业，中学时拖延的表白，好像只要不到最后一刻它就永远有理由不去做。而且还时常因为拖延让我愧疚背负巨大的负面心理而且我还发现它不是靠一时的奋发图强就能改变，这种惯性仿佛深入骨髓它就像我生命的一部分无法割舍。我意识到我之所以痛苦是因为我拖延的那一瞬间和现在反思的我根本不是同一个人！ 我对事物的权重比列根本不可靠它每时每刻都在变化拖延时它认为快乐最重要，到反思时它又认为成长更重要这根本不可理喻就像是左右脑互博要脱离这种痛苦只有消灭之中一个时态的权重现在的我肯定是认为成长更重要但我又无法将拖延那一时态彻底消灭这好像根本不是我能决定的，好像无路可走我只能和稀泥了也就是把他们平衡在一起，最大限度的放大现在时态权重在拖延时态权重出现时全力压制这需要一个在拖延出现时立马伴随它出现并更改权重的念头而且还要将反思权重这一视角提前切入不能等到事后那样就回天乏术了所以应该在拖延前或者拖延时就思考该行为会失去多么重要的东西，可是还有一个关键点我该怎么确保这两个念头会出现呢一味用力好像只会似的适得其反我需要穿越时间把力量送到未来该怎么办呢？我不是很确定我想到了一些不是很对的方法这还需要实践我认为拖延它是一种惯性就像破窗效应一样它会越来越强但当我和它反着来每一次成功时它都会变弱而我的能量会因为每一次成功变得更强大所以我的方案是当我处于反思权重时立马对拖延发起挑战没有任务时就即兴创造一个任务然后立马完成它先赢一次再说这样力量就会得到补充而且拖延也会越来越弱也就是说我认为只有把不拖延也培养成一种惯性才能成功这当然也需要毅力但每一次成功都是加油站这个想法还有待实践我不知道它是否正确。但我现在觉得我前面也说错了不应该加念头的回顾我以前的经验一旦给了思考的时间大脑总是能说服你做出拖延所以当我思考要不要拖延时注定会失败！我只有在拖延一开始就把它抹杀掉告诉他no!!! no!!! no!!!想都不要想去做只能有这一条指令不要给周旋的余地如果中途失败了那就再反思再斗争负面情绪没有任何作用只会浪费时间！
